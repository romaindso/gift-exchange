# Gift Exchange

## Demo
https://romaindso.github.io/gift-exchange/

## Features

* Vanilla JS only
* Redux for state management
* PureCSS for design
* Jest for testing


## Installation
Clone this project, then run in your console :
```bash
$ npm install
```

## Launch
Dev mode
```bash
$ npm start
```

Production mode
```bash
$ npm run build
```

## Test
Run tests
```bash
$ npm test
```
